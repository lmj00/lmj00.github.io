### 개요
RabbitMQ 문서의 “Manual Pages” 페이지는 최신 안정 릴리스에서 제공되는 매뉴얼 페이지를 섹션별로 나열한다. 이 페이지는 사용자가 참고할 수 있는 설정 파일과 운영 명령어를 한눈에 보여준다.

## Section 5: 파일 형식
Section 5에서는 환경 변수를 기반으로 한 설정 파일을 다룬다. 구체적으로 `rabbitmq-env.conf` 파일이 제시되며, 이 파일은 RabbitMQ 서버의 실행 환경을 환경 변수로 구성하는 데 사용된다. 매뉴얼 페이지 경로는 `man/rabbitmq-env.conf.5` 로 참조할 수 있다.

## Section 8: 시스템 관리 명령어
Section 8은 RabbitMQ 노드를 관리하고 운영하는 데 필요한 명령어들을 그룹화한다. 각 명령어는 특정 기능 영역을 담당하며, 다음과 같이 구분된다.

- `rabbitmqctl`: 일반적인 서비스 관리 및 운영자 명령어를 제공한다.  
- `rabbitmq-diagnostics`: 진단, 상태 점검 및 시스템 상태 관찰에 유용한 명령어를 포함한다.  
- `rabbitmq-plugins`: 플러그인 설치, 활성화, 비활성화 등을 관리한다.  
- `rabbitmq-upgrade`: 버전 업그레이드와 관련된 운영 작업을 수행한다.  
- `rabbitmq-queues`: 큐와 관련된 운영 작업을 처리한다.  
- `rabbitmq-streams`: 스트림과 관련된 운영 작업을 처리한다.  
- `rabbitmq-server`: RabbitMQ 서버 노드를 시작한다.  
- `rabbitmq-service`: Windows 환경에서 서비스를 관리한다.  
- `rabbitmq-echopid`: Windows 전용 유틸리티 도구로, 프로세스 ID를 출력한다.  

이들 명령어는 모두 `man/<command>.8` 형식의 매뉴얼 페이지로 제공된다.
