## stash
- 워킹 디렉토리에서 작업하던 내용을 stack에 보관한다.
```terminal
$ git stash
```

#### stack에 보관이 잘 되었는지 확인
`git stash list`


#### stack에 보관했던 내용 불러오기 
`git stash apply`


#### 왜 사용하는가?
- 브랜치에서 작업을 하다가 다른 브랜치로 가야 하는 상황으로 옮겨야 할 때 사용한다.
- 커밋을 하게 되면 필요 없는 커밋을 만들게 되기 때문에 stash를 사용한다.
